المحطة 2: القلب 🧠

لأنه بيعمل كل حاجة:
✅ تسجيل الطلبة | ✅ عرض المحتوى | ✅ متابعة الطلاب | ✅ امتحانات | ✅ تقارير

المحطة 3: الرموز 🧩

خريطة الأنظمة السريعة

•👉 CMS (المقرر) = إدارة كورس

•👉 LMS = إدارة التعلم بالكامل

•👉 LCMS = محتوى + تعلم

•👉 LAMS = أنشطة

•👉 Smart = ذكاء اصطناعي

المحطة 4: الوظائف (1) ⚙️

وظائف نظم إدارة التعلم (جزء 1)

1. تسجيل وإدارة الطلاب 👥

الطالب يعمل أكونت، يدخل الكورس، يتقسم جروبات.

📌 مثال: زي لما تدخلي جوجل كلاس روم بالكود.

2. إنشاء المحتوى 📚

المدرس يرفع: PDF، PowerPoint، فيديو. 👉 من غير تعقيد.

3. توصيل المحتوى 🚀

الطالب يوصله المحتوى بسهولة وسرعة.

4. إدارة الكورسات ✏️

تعديل / حذف / إضافة، تنظيم الدروس.

المحطة 5: الوظائف (2) 📢

وظائف نظم إدارة التعلم (جزء 2)

5. إدارة التعلم 🤝

تفاعل بين الطالب والمدرس والطلبة مع بعض.

📌 مثال: مناقشات أو كومنتات.

6. التواصل 📢

رسائل، منتديات، إعلانات.

7. التقييم 📝

امتحانات، كويزات، ملفات إنجاز.

8. التقارير 📊

متابعة درجات الطالب، مستواه، تقدمه.

المحطة 6: اللاعبون 👩‍💻

المستخدمين في النظام

👩‍🎓
الطالب

يدخل، يحل، يشارك.

👨‍🏫
المدرس

يشرح، يرفع، يمتحن.

👨‍💼
الإداري

ينظم، يدير الحسابات.

المحطة 7: المميزات ✅

مميزات نظم إدارة التعلم

✅ 1. متاحة في أي وقت (3 الفجر عادي 😅)
✅ 2. مرنة (مش مرتبطة بمكان)
✅ 3. الطالب هو المسؤول
✅ 4. عدد غير محدود (آلاف الطلبة)
✅ 5. مصادر كتير (فيديو+كتب)
✅ 6. تفاعل كبير
✅ 7. تعلم جماعي
✅ 8. تحليل بيانات
المحطة 8: التفصيل (CMS) 🎯

نظام إدارة المحتوى (CMS)

ده بيعمل إيه؟

1. إنشاء محتوى التعليمي (كائنات ومستودعات)
2. تخزينه
3. إعادة استخدامه
📌 مثال: تعملي درس وتستخدميه تاني في كورس تاني.
إنشاء المحتوى المستودع شخصنة المحتوى المستخدمين
المحطة 9: أنواع CMS 🛰️

تصنيفات إدارة المحتوى

1️⃣ البوابات الإلكترونية: مواقع إخبارية وجامعية.
2️⃣ المدونات: بلوجر ووردبريس.
3️⃣ المنتديات: للنقاش (زي زمان 😄).
4️⃣ المتاجر: جوميا وأمازون.
5️⃣ الويكي: مشاركة جماعية (ويكيبيديا).
المحطة 10: نظام المقرر 📈

Course Management System

هو أكثر تطور من CMS لأنه مش محتوى بس.

وضع المقرر أونلاين
تتبع تقدم الطالب
لوحة المناقشة
أدوات تواصل
❌ العيوب: مرونة قليلة، تفاعل محدود، مشاكل أمان، ضعف في الحفظ.
المحطة 11: مصنع المحتوى ⚒️

Learning Content Management System

مهامه الأساسية:
1. 📦 مستودع (Repository)
2. ✍️ إنشاء وتطوير المحتوى
3. 🔄 إعادة الاستخدام
