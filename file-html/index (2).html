🟡 المرحلة 1: تحديد الجمهور المستهدف

📌 دي أهم خطوة في اللعبة!

لأن كل حاجة بعدها بتتبني عليها.

✳️ بيتم فيها إيه؟ (غرفة الاجتماعات)

اجتماع فريق العمل (مخرج – معد – مذيع…) والبحث عن:

  • مين الجمهور بالظبط؟
  • إيه الهدف النهائي؟
  • إيه القالب المناسب ليهم؟

👥 نوع الجمهور:

أ) جمهور عام: كل الناس (بدون قيود).

ب) جمهور خاص: (سني - مهني - جغرافي).

مثال: طفل مدرسة vs طفل قبل المدرسة / ربة منزل vs عاملة.

⏰ الوقت واللغة:

📌 التوقيت: لازم وقت ذروة (Prime Time) للجمهور.

📌 اللغة: (بسيطة للعامة / أرقى للمثقفين / سهلة جداً للأطفال).

📌 القالب: (حوار - تحقيق - مجلة - مسلسل) حسب ذوقهم.

🟡 المرحلة 3: الإعداد (خلف الكواليس)

1️⃣ التخطيط (خريطة العمل):

تقسيم الموضوع ← أسئلة ← توزيع الوقت ← اختيار المصادر.

2️⃣ المصادر:

شخصية: لقاءات مباشرة.

مكتبية: أبحاث، جرائد، أرشيف.

3️⃣ جمع المادة:

بحث عميق ← تحديد ضيوف ← وضع بدائل للطوارئ.

5️⃣ المؤثرات الصوتية:

طبيعية (واقعية) / صناعية (مسجلة). تعطي مصداقية وإحساس بالمكان.

6️⃣ الموسيقى:

تتر (بداية ونهاية) / ربط (بين الفقرات). لازم تليق بموود البرنامج.

🔴 المرحلة 4: التنفيذ (Action)

📜 الاسكريبت (قلب البرنامج)

  • ✨ افتتاحية ونهاية تشد.
  • ✨ بداية قوية (تصريح أو مؤثر).
  • ✨ وحدة البرنامج (ترابط العناصر).
  • ✨ كلمات بسيطة تناسب الجمهور.

🎧 المخرج في الاستوديو:

يتأكد من الأجهزة ← يطمن الضيوف ← يراقب الوقت ← تسجيل البيانات بعد انتهاء الشريط.

✂️ المونتاج والمزج:

حذف الأخطاء ← تحسين الجودة ← دمج الصوت مع الموسيقى والمؤثرات باحترافية.

🟣 التقييم (مرحلة الـ Score)

1) تقييم مرحلي:

آراء الجمهور أثناء العرض (مكالمات/رسائل) لقياس الفهم والجذب.

2) تقييم نهائي:

شكلي: (قالب، مذيع، مونتاج).
مضمون: (هل وعي الجمهور زاد؟ هل اتغير سلوكه؟).

🏁 ملخص المهمة (الزتونة)

  • 🎯 الإنتاج 4 خطوات: جمهور، فكرة، إعداد، تنفيذ.
  • 🎯 الجمهور هو البوصلة اللي بتحرك كل حاجة.
  • 🎯 الإعداد هو الشغل التقني والمكتبي قبل المايك.
  • 🎯 التنفيذ هو الاسكريبت والتسجيل والمونتاج.
